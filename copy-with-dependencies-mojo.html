devsak:copy-with-dependencies

Full name:

com.github.jdelker:devsak-maven-plugin:2.1:copy-with-dependencies

Description:

Goal that copies an artifact, including its dependencies from the repository to a defined location.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.
  • Binds by default to the lifecycle phase: prepare-package.

Required Parameters

Name Type Since Description
<markersDirectory> File 1.0 Directory to store marker files
Default value is: ${project.build.directory}/.markers.
<outputDirectory> File 1.0 Directory to store marker filesF
Default value is: ${project.build.directory}/copied-artifacts.

Optional Parameters

Name Type Since Description
<artifactItems> List 1.0 Collection of ArtifactItems to work on. See Usage for details.
<artifactTracking> boolean 1.0 Track installed artifacts, so they are not copied on any subsequent build.
Default value is: true.
