devsak:unpack

Full name:

com.github.jdelker:devsak-maven-plugin:2.1:unpack

Description:

Goal for unpacking files

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: process-sources.

Required Parameters

Name Type Since Description
<fileSet> FileSet - The list of class or jar files.
<outputDirectory> File - Location of the output.
Default value is: ${project.build.directory}/unpacked.
Alias is: outputDir.

Optional Parameters

Name Type Since Description
<excludes> String[] - (no description)
<includes> String[] - The list of class files to process.

Parameter Details

(no description)
