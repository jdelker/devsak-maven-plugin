devsak:upload

Full name:

com.github.jdelker:devsak-maven-plugin:2.1:upload

Description:

Uploads file to remote repository.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: deploy.

Optional Parameters

Name Type Since Description
<file> File - The path to the file to be uploaded.
User property is: upload.file.
<fileSet> FileSet - Alternate parameter to specify a set of files to upload.
<headers> Map - Custom HTTP headers to add to each request.
<ignoreMissingFile> boolean - If true, do not fail build when file is missing.
Default value is: false.
User property is: upload.ignoreMissing.
<preemptiveAuth> boolean - Set to true if the server requires credentials in the initial request.
Default value is: false.
User property is: upload.preemptiveAuth.
<serverId> String - The server Id in settings.xml with credentials to use.
User property is: upload.serverId.
<serverPath> String - The server path where the file will be uploaded, ie path/to/file.ext. Will be appended to the serverUrl parameter.
Default value is: /.
User property is: upload.serverPath.
<serverUrl> String - The base URL of the server, ie http://server.example.com/.
User property is: upload.serverUrl.
<session> MavenSession - (no description)
User property is: session.
<skip> boolean - Set to true to skip execution.
Default value is: false.
User property is: upload.skip.
<usePOST> boolean - Use POST instead of PUT
Default value is: false.
User property is: upload.post.
